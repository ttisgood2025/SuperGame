# 萌宠三叠消消乐（Cocos Creator 3.8.2）

这是一个针对 **Cocos Creator 3.8.2** 的三叠消除类小游戏策划与代码骨架，主题为萌宠 2D 风格，共 60 关，关卡型推进，并包含可持续游玩的上瘾循环机制。

## 核心卖点
- **三叠消除 + 轻策略**：玩家点击上层宠物卡牌，卡牌进入底部槽位，凑齐 3 张同图案即消除。
- **60 关关卡推进**：分为 4 个章节，难度递增，包含障碍、限步、目标类型变化。
- **上瘾循环机制**：短局（1-3 分钟）→ 奖励反馈 → 轻成长（皮肤、家园装饰）→ 再挑战。
- **萌宠 2D 美术风格**：猫、狗、兔、熊猫等宠物头像，圆角、柔色、高反馈特效。

## 目录
- `docs/game-design.md`: 完整玩法与系统设计。
- `config/levels.json`: 60 关配置样例（可直接扩展到生产）。
- `src/GameConfig.ts`: 关卡结构、参数配置定义。
- `src/LevelGenerator.ts`: 基于关卡参数生成牌堆数据的逻辑示例。

## 快速接入 Cocos Creator 3.8.2
1. 在 Cocos Creator 3.8.2 新建 2D 项目。
2. 将 `src/` 下脚本拷贝到项目 `assets/scripts/`。
3. 将 `config/levels.json` 作为 `TextAsset` 或远程配置加载。
4. 用预制体（PetTile）绑定宠物图标与点击事件。
5. 在主场景挂载一个 `GameController`（可按 `docs/game-design.md` 的流程实现）。
